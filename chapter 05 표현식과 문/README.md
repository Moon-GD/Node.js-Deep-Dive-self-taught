## 4. 문
- 최소 실행 단위로서, 문이 모여 프로그램 전체 혹은 일부를 구성
- 토큰(문법적인 최소 단위)이라고 하는 단위로 나눌 수 있음
    - Ex : '''const a = 1''' => '''const, a, "=", 1로 분리 가능'''
- 할당문, 조건문, 반복문 등이 존재
프로그램의 구성 단위, 즉, 문이 모여 프로그램 혹은 일부를 구성

## 5. 세미 콜론
