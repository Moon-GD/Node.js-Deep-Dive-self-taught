#### 1️⃣ let : 분리되어 있는 변수 선언과 초기화 단계
- var로 선언된 변수는 선언과 초기화(undefined)가 함께 이루어짐
- let으로 선언한 변수는 JS 엔진에 의해 런타임 이전에 실행되긴 하지만 변수 선언문에 도달했을 때 초기화가 진행
- 따라서, '스코프 시작 지점 ~ 초기화가 진행되기 이전 지점'을 Temporal Dead Zone (TDZ)라고 부른다
